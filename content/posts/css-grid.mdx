# CSS Grid 网格布局指南：同时管理行与列

你想要一个三列等宽布局：

```css
/* Flexbox 方案 */
.container {
  display: flex;
}
.item {
  flex: 1;
}
```

如果第二行要两列、第三行要一列，Flexbox 也能通过换行和宽度计算实现，但行与列之间的关系会越来越难维护。Grid 更适合直接描述这种二维结构。

可以先把容器划分为六个轨道，再让不同项目跨越不同数量的轨道：

```css
.container {
  display: grid;
  grid-template-columns: repeat(6, 1fr);
}

.item:nth-child(-n + 3) {
  grid-column: span 2; /* 第一行三列 */
}

.item:nth-child(4),
.item:nth-child(5) {
  grid-column: span 3; /* 第二行两列 */
}

.item:nth-child(6) {
  grid-column: span 6; /* 第三行一列 */
}
```

Grid 是二维布局：同时控制行和列。Flexbox 是一维布局：只控制一行或一列。

---

## 1. 核心概念

### 1.1 容器和项目

- **容器（grid container）**：设置 `display: grid` 的元素
- **项目（grid item）**：容器的直接子元素

### 1.2 行和列

- **行（row）**：水平方向
- **列（column）**：垂直方向

### 1.3 网格线

- 行和列的交界处形成网格线
- 网格线从 1 开始编号

---

## 2. 容器属性

### 2.1 grid-template-columns / grid-template-rows

**定义列宽和行高。**

```css
.container {
  /* 固定宽高 */
  grid-template-columns: 200px 100px 200px;
  grid-template-rows: 50px 50px;

  /* repeat 简化重复 */
  grid-template-columns: repeat(3, 200px);
  grid-template-rows: repeat(2, 50px);

  /* fr 单位：等分 */
  grid-template-columns: 1fr 2fr 1fr; /* 中间列是两边的两倍 */

  /* auto-fill：自动填充 */
  grid-template-columns: repeat(auto-fill, 200px);

  /* minmax：范围 */
  grid-template-columns: 1fr 1fr minmax(300px, 2fr);

  /* auto：自动 */
  grid-template-columns: 100px auto 100px;
}
```

### 2.2 gap

**设置间距。**

```css
.container {
  gap: 10px 20px;        /* 行间距 10px，列间距 20px */
  row-gap: 10px;         /* 行间距 */
  column-gap: 20px;      /* 列间距 */
}
```

### 2.3 grid-template-areas

**定义区域。**

```css
.container {
  grid-template-areas:
    "header header header"
    "sidebar content content"
    "footer footer footer";
}

.header {
  grid-area: header;
}
.sidebar {
  grid-area: sidebar;
}
.content {
  grid-area: content;
}
.footer {
  grid-area: footer;
}
```

### 2.4 grid-auto-flow

**自动布局顺序。**

```css
.container {
  grid-auto-flow: row;         /* 先行后列（默认） */
  grid-auto-flow: column;      /* 先列后行 */
  grid-auto-flow: row dense;   /* 先行后列，尽可能填满 */
  grid-auto-flow: column dense;
}
```

### 2.5 justify-items / align-items

**项目在单元格内的对齐。**

```css
.container {
  justify-items: start;   /* 左对齐 */
  justify-items: center;  /* 水平居中 */
  justify-items: end;     /* 右对齐 */
  justify-items: stretch; /* 拉伸填满（默认） */

  align-items: start;     /* 顶部对齐 */
  align-items: center;    /* 垂直居中 */
  align-items: end;       /* 底部对齐 */
  align-items: stretch;   /* 拉伸填满（默认） */
}
```

### 2.6 justify-content / align-content

**整个网格在容器内的对齐。**

```css
.container {
  justify-content: start;        /* 左对齐 */
  justify-content: center;       /* 水平居中 */
  justify-content: end;          /* 右对齐 */
  justify-content: space-between;
  justify-content: space-around;
  justify-content: space-evenly;

  align-content: start;          /* 顶部对齐 */
  align-content: center;         /* 垂直居中 */
  align-content: end;            /* 底部对齐 */
}
```

## 3. 项目属性

### 3.1 grid-column / grid-row

**指定项目占据的网格线。**

```css
.item {
  grid-column-start: 1;
  grid-column-end: 3;
  /* 简写 */
  grid-column: 1 / 3;

  grid-row-start: 1;
  grid-row-end: 2;
  /* 简写 */
  grid-row: 1 / 2;
}
```

**span 关键字：**

```css
.item {
  grid-column: span 2; /* 占两列 */
  grid-row: span 3;    /* 占三行 */
}
```

### 3.2 grid-area

**指定项目所在的区域。**

```css
.item {
  grid-area: header;
}

/* 或直接指定网格线 */
.item {
  grid-area: 1 / 1 / 3 / 3; /* row-start / col-start / row-end / col-end */
}
```

### 3.3 justify-self / align-self

**单个项目的对齐。**

```css
.item {
  justify-self: center;
  align-self: center;
}
```

## 4. 常见布局模式

### 4.1 三列等宽

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}
```

### 4.2 固定侧边栏 + 自适应内容

```css
.container {
  display: grid;
  grid-template-columns: 200px 1fr;
}
```

### 4.3 圣杯布局

```css
.container {
  display: grid;
  grid-template-areas:
    "header header header"
    "sidebar content ads"
    "footer footer footer";
  grid-template-columns: 200px 1fr 200px;
  grid-template-rows: 60px 1fr 60px;
}
```

### 4.4 响应式网格

```css
.container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  gap: 20px;
}
```

**效果：** 列宽至少 200px，容器变宽时自动增加列数。

### 4.5 瀑布流布局

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-auto-rows: 10px;
}

.item:nth-child(1) {
  grid-row: span 10;
}
.item:nth-child(2) {
  grid-row: span 15;
}
.item:nth-child(3) {
  grid-row: span 8;
}
```

## 5. Grid vs Flexbox

| 对比项 | Grid | Flexbox |
|--------|------|---------|
| 维度 | 二维（行 + 列） | 一维（行或列） |
| 适用场景 | 复杂布局、页面骨架 | 简单布局、组件内部 |
| 对齐方式 | 行列都可对齐 | 主轴或交叉轴对齐 |
| 项目大小 | 可以精确控制 | 自动分配 |

**Flexbox 和 Grid 放在一起看：**

- 页面整体布局 → Grid
- 组件内部布局 → Flexbox
- 需要精确控制行列 → Grid
- 简单的一行/一列 → Flexbox

## 6. 注意事项

### 6.1 隐式网格

当项目超出定义的行列时，会自动创建隐式网格：

```css
.container {
  grid-template-columns: 200px 200px; /* 只定义两列 */
  grid-auto-columns: 100px;           /* 超出的列宽 100px */
}
```

### 6.2 网格重叠

多个项目占据同一单元格时，用 `z-index` 控制层级：

```css
.item1 {
  grid-column: 1 / 3;
  z-index: 1;
}
.item2 {
  grid-column: 2 / 4;
  z-index: 2;
}
```

### 6.3 fr vs %

- `fr`：分配剩余空间
- `%`：相对于容器宽度

```css
grid-template-columns: 1fr 1fr;  /* 两列等分剩余空间 */
grid-template-columns: 50% 50%;  /* 两列各占容器的一半 */
```

## 7. 回到布局选择

**容器属性：**

| 属性 | 作用 |
|------|------|
| grid-template-columns | 列宽 |
| grid-template-rows | 行高 |
| gap | 间距 |
| grid-template-areas | 区域 |
| grid-auto-flow | 自动布局顺序 |
| justify-items / align-items | 项目对齐 |
| justify-content / align-content | 网格对齐 |

**项目属性：**

| 属性 | 作用 |
|------|------|
| grid-column / grid-row | 占据的网格线 |
| grid-area | 区域 |
| justify-self / align-self | 单个项目对齐 |

**常用值：**

- `fr` — 等分空间
- `repeat(n, x)` — 重复 n 次
- `auto-fit` — 自动填充
- `minmax(min, max)` — 范围
- `span n` — 占 n 格
